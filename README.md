# Проект "My App"

## Описание

В проекте реализован интернет-магазин. Приложение реализовано через Django. Проект будет дополняться.

### Описание модулей

В пакете **config** реализован проект и необходимые конфигурационные файлы для него.

В пакете **catalog** расположено само приложение, а также файлы с маршрутизацией, с контроллерами, миграциями и др. необходимыми файлами. 

В файле **templates** находятся html шаблоны для страниц приложения.

В пакете **static** лежат статические файлы, такие как стили, изображения и скрипты.

В файле **manage** находится код для управления проектом в Django.

В пакете **management** есть пакет **commands** в котором присутствуют кастомные команды.

Также, в корне проекта расположен файл фикстур **catalog_fixture.json**.

В файле **.env.example** хранятся шаблоны чувствительных данных. Создайте файл .env из копии этого файла и замените значения переменных реальными данными.

В файле **forms** добавлены формы для создания или редактирования продуктов с примененной валидацией.

В пакете **teplatetags** располагается файл с фильтром для медиа файлов товаров.

В пакете **blog** расположено второе приложение для ведения блога, а также файлы с маршрутизацией, с контроллерами, миграциями и др. необходимыми файлами.

В пакете **users** находится приложение, управляющее пользователями. Реализованы представления для регистрации пользователя, для логина и логаута, а также для регистрации профиля пользователя. 
При успешной регистрации на почту отправляется приветственное письмо. 
Представления для просмотра, редактирования и удаления продуктов и статей, а также для их создания ограничены в доступе - доступны только для зарегестрированных пользователей.

### Установка

1. Клонируйте репозиторий

```commandline
git clone https://github.com/mariazhiganova/django_app.git

```

2. Установите зависимости

```commandline
poetry install
```
